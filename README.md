# ReactRunning

Next.js를 사용한 로그인 페이지 프로젝트입니다.

## 기능

- 이메일과 비밀번호를 사용한 로그인
- 반응형 디자인
- 모던한 UI/UX

## 설치 및 실행

1. 프로젝트 클론
```bash
git clone https://github.com/jigookang/ReactRunning.git
```

2. 의존성 설치
```bash
npm install
```

3. 개발 서버 실행
```bash
npm run dev
```

4. 브라우저에서 확인
```
http://localhost:3000/login
```
